# Arsd-login ERP - Responsive

## Usage

### Run these commands in VS-Code terminal before using given files:-

To open terminal in VS-code : Ctrl + ~

```

npm init -y

npm install -D tailwindcss

npx tailwindcss init

npm run build #


```
